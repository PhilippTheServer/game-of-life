# About

Game of life, naive implementation. 
Uses the terminal as a canvas.

# Prerequisits

Make sure you have an enviornment where the following dependencies are installed:
- cmake
- g++ (Version 23)

# Build
The executable can only be build for linux and macos using these commands:

```sh
cmake -S . -B build
cmake --build build 
```

# Execute
To run the binary:

```sh
./build/game-of-life
```
